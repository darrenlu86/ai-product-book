
# 前言

嗨，我是紹民。

你手上這本書，是寫給那些「想做點什麼，但不知道從哪裡開始」的人。

我是一個產品經理，在台灣的網路認證公司工作。我不是工程師，大學時代的程式作業有一半是跑不動的。但在過去這段時間，我用 AI 做了十六個以上的 Side Project — 從婚禮邀請函到 SaaS 產品，有些真的有人在用，有些有在收費。

這件事在幾年前是不可能的。

這本書不是要教你寫程式。市面上教寫程式的書已經夠多了。這本書要給你的，是一張地圖 — 讓你知道一個軟體產品是怎麼組成的、每個部分負責什麼、你該在什麼時候做什麼決定。

有了這張地圖，你就知道怎麼指揮 AI 幫你做事。

書裡穿插了不少我自己的經驗，包括踩過的坑、犯過的蠢、半夜被 API 搞到崩潰的時刻。這些都是真的。我覺得誠實地分享失敗比假裝一切順利更有用。

如果你讀完這本書，對軟體產品有了基本的認識，知道怎麼跟 AI 合作把想法變成現實 — 那就值了。

紹民
2026 年，台北
